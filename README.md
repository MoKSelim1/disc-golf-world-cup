Disc Golf World Cup

Static tournament management app for the World Cup style disc golf event at North Park.

## Development

```bash
cd disc-golf-tournament
npm install
npm run dev
```

## Build

```bash
cd disc-golf-tournament
npm run build
```

The admin passphrase is defined in `src/components/admin/PassphraseGate.tsx`.
The default v1 passphrase is `northpark`.
